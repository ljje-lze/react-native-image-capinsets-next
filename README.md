## 说明

设置图像拉伸时可调整大小的图像选项。拉伸对拖拽缩略图以及占位图有效。基于 [Image resizable](https://developer.huawei.com/consumer/cn/doc/harmonyos-references-V5/ts-basic-components-image-V5#resizable11) 属性实现。
